# CRUD Eventos

Aplicação Java com PostgreSQL para gerenciamento de eventos.

## Funcionalidades
- Cadastro de usuários
- Cadastro de participantes
- Cadastro de organizadores
- Cadastro de locais
- Cadastro de eventos
- Cadastro de pedidos
- Cadastro de pagamentos

## Tecnologias
- Java 21
- PostgreSQL
- JDBC
- Maven
